# Crime Reporting System Mockup

This is a frontend mockup for a Crime Reporting System built with React, Vite, Tailwind CSS, and Shadcn UI.

## Features

- **User Reporting**: Interactive map to pin crime locations and submit details.
- **Admin Dashboard**: View list of reports and a map overview of all incidents.
- **Map Integration**: Uses Leaflet and OpenStreetMap.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at `http://localhost:5173`.

## Navigation

- **Home (/)**: User Report Page
- **Admin (/admin)**: Admin Dashboard
